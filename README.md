# tavphive

Module **billing SaaS** untuk TAVP. Buat yang mau bikin produk berlangganan.

## Fitur

- Subscription / langganan
- Invoicing / tagihan
- Manajemen paket (plan)
- Webhook handling
- Payment gateway:

| Gateway | Wilayah |
|---|---|
| Stripe | Internasional |
| PayPal | Internasional |
| Midtrans | Indonesia |
| Xendit | Indonesia |
| mayar.id | Indonesia |
| Duitku | Indonesia |
| Flip Business | Indonesia |
| iPaymu | Indonesia |

## Cara pakai (rencana)

```
tavp module:install tavp/tavphive
```
